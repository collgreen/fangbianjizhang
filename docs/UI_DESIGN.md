# UI/UX 规范

## 1. 导航结构

底部导航栏 4 个 Tab + 中央悬浮 FAB：

```
┌────────────────────────────────┐
│                                │
│         页面内容区域            │
│                                │
├────┬────┬──────┬────┬─────────┤
│ 首页│资产│ [＋] │统计│  设置   │
└────┴────┴──────┴────┴─────────┘
```

- FAB 按钮突出显示（主题色填充，圆形，48dp+）
- 当前 Tab 图标高亮，其余灰色
- 页面切换无动画（即时切换）

## 2. 主题配色

基于 Material 3 动态主题：

| 语义 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 收入金额 | #4CAF50 (绿) | #81C784 |
| 支出金额 | #F44336 (红) | #E57373 |
| 主题色 | #1976D2 (蓝) | #64B5F6 |
| 背景 | #FAFAFA | #121212 |
| 卡片背景 | #FFFFFF | #1E1E1E |
| 警告色 | #FF9800 (橙) | #FFB74D |

支持三种主题：浅色 / 深色 / 跟随系统。

## 4. 记账页面

点击 FAB 进入，全屏页面：

```
┌────────────────────────────┐
│ ✕                    保存   │
├────────────────────────────┤
│ [支出] [收入] [转账] [借贷] │  ← 顶部 Tab
├────────────────────────────┤
│         ¥ 0.00             │  ← 金额显示区
├────────────────────────────┤
│ 分类选择（网格）            │
│ 🍜餐饮 🏠生活 🎮娱乐       │
│ 🚇交通 📚教育 ···          │
├────────────────────────────┤
│ 选择二级: 早餐 午餐 晚餐    │  ← 选中一级后展开
├────────────────────────────┤
│ 账户: 招商银行储蓄卡  ▼     │
│ 日期: 2026-02-20     ▼     │
│ 备注: _______________       │
│ 📷 添加图片                 │
├────────────────────────────┤
│  ┌─┬─┬─┬──┐               │
│  │7│8│9│删│               │  ← 数字键盘
│  │4│5│6│+ │               │
│  │1│2│3│- │               │
│  │.│0│  │完│               │
│  └─┴─┴──┴──┘               │
└────────────────────────────┘
```

- 分类图标网格：每行 5 个，可滚动
- 数字键盘内置，支持简单加减运算
- 转账模式：隐藏分类，显示源账户和目标账户
- 借贷模式：增加对方名称和预计还款日字段

## 8. 通用组件规范

| 组件 | 规格 |
|------|------|
| 触摸目标 | 最小 48dp × 48dp |
| 卡片圆角 | 12dp |
| 内边距 | 页面 16dp，卡片内 12dp |
| 字体 | 金额 20sp 粗体，正文 14sp，辅助 12sp |
| 图标 | Material Icons，24dp |
| 分割线 | 1dp，颜色 #E0E0E0 / #333333(深色) |
| 空状态 | 居中插图 + 引导文字 |
| 加载状态 | CircularProgressIndicator 居中 |
